# 프로젝트 설정

## 자동 활성화 기능

이 프로젝트에서는 다음 기능이 자동으로 활성화됩니다:

### 1. 토큰 모니터링
- 세션 중 주기적으로 남은 토큰 수를 체크합니다
- 작업 완료 시점에 커밋/배포를 위한 충분한 토큰이 남았는지 확인합니다
- 토큰이 30,000 이하로 떨어지면 사용자에게 "커밋/배포하기 좋은 타이밍"임을 알립니다

### 2. 작업 완료 시 알림
- 의미있는 작업이 완료되면 커밋/배포 여부를 확인합니다
- 여유 토큰이 있을 때 미리 알려줍니다

## Claude Code에게

**중요**: 이 프로젝트에서 작업할 때는 항상:
1. 세션 시작 시 토큰 모니터링 모드를 활성화하세요
2. 작업이 어느 정도 진행되었을 때 (토큰이 50,000 정도 남았을 때) 사용자에게 커밋/배포 타이밍을 알려주세요
3. 토큰이 30,000 이하로 떨어지면 반드시 경고하세요
